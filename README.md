# Rock Nitrogen Analysis

This repository contains the analysis of nitrogen concentrations in rock samples from Sedgwick and Santa Cruz Island. 

## Files

- `rock_n_visuals.Rmd` – R Markdown file containing the analysis and code.
- `rock_n_visuals.html` – HTML output of the analysis (click the link below to view in browser).

## View HTML Output

[View Rock Nitrogen Analysis HTML](rock_n_visuals.html)

## Data
